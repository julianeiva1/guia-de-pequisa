# Guia de Pesquisa - Acesso ao Conhecimento

Uma landing page moderna e responsiva para apresentar o **Guia de Pesquisa**, um material educativo que ajuda estudantes e pesquisadores a conduzirem pesquisas de forma estruturada e profissional.

## O que é?

O **Guia de Pesquisa** é uma plataforma digital que oferece orientações práticas para transformar uma simples ideia em um trabalho de pesquisa bem fundamentado. A landing page apresenta o projeto, seus benefícios e histórias de sucesso de usuários.

## Principais Funcionalidades

- Header fixo com navegação e menu responsivo para mobile
- Design moderno com 6 cores vibrantes e animações suaves
- Carousel de feedbacks com suporte a clique, drag e toque
- 100% responsivo (desktop, tablet, mobile)
- Animações de scroll que revelam elementos conforme você rola a página
- 6 seções principais: Hero, Sobre, Conteúdo, Feedbacks, Contato e CTA

## Tecnologias

- HTML5
- CSS3
- JavaScript ES6+

## Como Usar

### Clonar e Abrir
```bash
git clone https://github.com/julianeiva1/guia-de-pequisa.git
cd guia-de-pequisa
```

Abra `index.html` no navegador ou use um servidor local:
```bash
python -m http.server 8000
```

### Editar Conteúdo
- Textos e estrutura: `index.html`
- Cores e estilos: `styles.css` (variáveis CSS no `:root`)
- Comportamentos: `script.js`
- Imagem: substitua `assets/guia-cover.jpg`

## Estrutura

```
├── index.html      # Página principal
├── styles.css      # Estilos e design system
├── script.js       # Interatividade (carousel, menu, animações)
├── assets/         # Imagens
└── README.md       # Este arquivo
```
